## 需求
### 需求:已登录用户可以发布短帖
系统必须允许已登录用户发布文本帖子，并将帖子与当前登录用户关联。

#### 场景:成功发布帖子
- **当** 已登录用户提交符合规则的帖子内容
- **那么** 系统创建新帖子并在后续时间线中显示该帖子

### 需求:帖子内容必须经过服务端校验
系统必须在服务端校验帖子内容不能为空且不能超过允许长度。

#### 场景:帖子内容为空
- **当** 已登录用户提交空内容帖子
- **那么** 系统必须拒绝请求并返回校验错误

#### 场景:帖子内容超长
- **当** 已登录用户提交超过长度限制的帖子内容
- **那么** 系统必须拒绝请求并返回校验错误
