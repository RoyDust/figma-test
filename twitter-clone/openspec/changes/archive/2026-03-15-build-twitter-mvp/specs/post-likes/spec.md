## 新增需求

### 需求:已登录用户可以点赞帖子
系统必须允许已登录用户对帖子创建点赞关系。

#### 场景:成功点赞
- **当** 已登录用户对尚未点赞的帖子提交点赞请求
- **那么** 系统创建点赞记录并增加该帖子的点赞数量

### 需求:用户不能重复点赞同一帖子
系统必须禁止同一用户对同一帖子创建重复点赞记录。

#### 场景:重复点赞请求
- **当** 已登录用户对已点赞帖子再次发送点赞请求
- **那么** 系统必须避免创建重复记录并返回可预期结果

### 需求:已登录用户可以取消点赞
系统必须允许已登录用户移除自己对帖子的点赞关系。

#### 场景:成功取消点赞
- **当** 已登录用户对已点赞帖子提交取消点赞请求
- **那么** 系统移除点赞记录并减少该帖子的点赞数量
